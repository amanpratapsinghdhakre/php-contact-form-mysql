# PHP Contact Form with MySQL

This is a simple full-stack project built using PHP and MySQL.  
The application allows users to send messages through a contact form which are stored in a database.

## Features

- Responsive contact form
- Form validation
- Data stored in MySQL database
- Admin page to view submitted messages
- Clean CSS design

## Technologies Used

- PHP
- MySQL
- HTML
- CSS

## Project Structure

php-contact-form-mysql
│
├── index.php
├── contact.php
├── admin.php
├── db.php
└── style.css

## How to Run the Project

1. Install XAMPP
2. Start Apache and MySQL
3. Place the project folder inside the htdocs directory
4. Import the database in phpMyAdmin
5. Open the project in browser:
http://localhost/php-contact-form-mysql
